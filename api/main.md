# API Spec

**Libraries:**

* `flask` as the underlying python web framework
* `marshmallow` for form schema validation
* `peewee` as the SQL schema definition

**Endpoints:**

* /[data](data.md)/ `POST, GET, DELETE`
* /data/targets/ `GET`
* /data/units/ `GET`
* /data/sensors/ `GET`
* 
* /devices/ `POST, GET, UPDATE, DELETE`
